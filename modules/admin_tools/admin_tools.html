Cronjob manuell ausführen

Der Cronjob ist ein Skript, das üblicherweise jede Nacht ausgeführt wird. Es führt Aufräumarbeiten im Stubegru System durch. Der Cronjob wird vom Server-Admin konfiguriert. Hier kann der Cronjob manuell ausgeführt werden. Dies kann inkonsistente Datensätze verursachen, wenn gerade Nutzer in Stubegru angemeldet sind. Bitte nur benutzen, wenn du weißt, was du tust...
